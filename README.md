# Time Logs

A simple FullStack application to insert & retrieve time logs

- Looks simple & functional
- Performs well
- Responsive
- Unit tests

The app is divided into 2 parts:
## Server
Technologies used:
- Node.js
- PostgresSQL
- Sequelize
- Heroku

## Client
Technologies used:
- ReactJS
- Material-Ui
- Axios
- Netlify
